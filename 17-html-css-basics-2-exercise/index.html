Why Learn HTML?

HTML is the foundation of all web pages.
Without HTML, you wouldn't be able to
organise text or add images or videos to your
web pages.
HTML is the beginning of everything you need

Why Learn CSS?

Without CSS, every web page would be drab
plain text and images that flowed straight
down the page.
With CSS, you can add colour and background
image and change the layout of your page -
your web pages can feel like works of art!
